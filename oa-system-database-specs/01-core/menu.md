# 菜单管理数据库设计

> **对应前端规范**: [menu_Technical.md](../../../oa-system-frontend-specs/core/menu/menu_Technical.md)
> **数据库**: MySQL 8.0+
> **表前缀**: `sys_`
> **版本**: v1.0.0

---

## 数据库表设计

完整的SQL已在前端规格中定义,包含18个预置菜单项。

### 菜单层级结构

```
一级菜单 (5个)
├── 系统管理 (3个子菜单)
├── 员工管理 (1个子菜单)
├── 资产管理 (3个子菜单)
├── 审批管理 (3个子菜单)
└── 会议管理 (3个子菜单)
```

### 权限控制

- 菜单权限通过用户角色关联
- 支持动态菜单加载
- 支持按钮级权限控制
