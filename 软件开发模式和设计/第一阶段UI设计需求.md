# OA系统第一阶段 UI设计需求文档

## 设计目标
为OA系统的核心基础模块创建直观、易用的用户界面,包含员工管理、部门架构和公共资源管理三大功能模块。

---

## 模块一: 员工名录管理系统

### 页面1: 员工列表页

**布局结构:**
- 顶部: 页面标题 "员工名录" + 操作按钮区
- 左侧: 筛选面板 (可折叠)
- 中间: 数据表格
- 右侧: 快捷统计面板

**功能区域:**

**1. 顶部操作区**
- 主按钮: "新增员工" (蓝色,主要操作)
- 次要按钮: "批量导入" "导出列表"
- 搜索框: 支持按姓名/工号/手机号搜索
- 视图切换: 表格视图 / 卡片视图

**2. 左侧筛选面板**
- 员工状态: 在职 / 离职 / 停薪留职 (单选)
- 部门筛选: 下拉多选树形结构
- 试用期状态: 试用期内 / 已转正
- 职位级别: 下拉筛选
- 性别: 男 / 女
- 入职时间范围: 日期选择器

**3. 数据表格 (表格视图)**
列设置:
| 列名 | 宽度 | 说明 |
|------|------|------|
| 选择 | 50px | 复选框 |
| 头像 | 60px | 圆形头像缩略图 |
| 员工编号 | 120px | 超链接,点击查看详情 |
| 姓名 | 100px | 显示头像+姓名 |
| 部门 | 150px | 显示部门名称 |
| 职位 | 120px | 职位名称 |
| 联系电话 | 130px | 可点击拨打 |
| 邮箱 | 200px | 可点击发送邮件 |
| 试用期状态 | 120px | 彩色标签 (试用期内=黄色,已转正=绿色) |
| 员工状态 | 100px | 彩色标签 (在职=绿色,离职=灰色) |
| 入职日期 | 120px | YYYY-MM-DD格式 |
| 操作 | 150px | 编辑/更多操作菜单 |

表格功能:
- 支持列排序
- 支持列显示/隐藏配置
- 支持表格数据分页 (每页20/50/100条)
- 行点击高亮

**4. 右侧统计面板**
卡片式统计:
- 总员工数: 数字显示
- 在职人数: 数字 + 绿色趋势箭头
- 试用期人数: 数字 + 黄色标签提醒
- 本月新入职: 数字
- 部门分布: 简易饼图

**卡片视图 (备选):**
- 每个卡片显示员工头像、姓名、职位、部门、工号
- 卡片大小: 200x280px
- 卡片间距: 20px
- 支持瀑布流布局

### 页面2: 员工详情页

**页面布局:**
- 左侧: 员工基本信息卡片 (固定)
- 右侧: 详细信息标签页

**左侧信息卡片:**
1. 头像区: 大尺寸圆形头像 (150x150px) + "更换头像"按钮
2. 基本信息:
   - 员工编号 (大字体显示)
   - 姓名 (大字体加粗)
   - 职位 (标签样式)
   - 部门 (链接,点击跳转部门详情)
   - 员工状态: 彩色标签

**右侧标签页:**
标签1: "基本信息"
- 表单式布局,两列
- 字段: 英文名、性别、出生日期、联系电话、邮箱、办公位置、紧急联系人
- 所有字段可编辑,保存按钮在底部

标签2: "工作信息"
- 字段: 入职日期、试用期状态、转正日期、直属上级 (下拉选择员工)、工龄 (自动计算显示)
- 编辑按钮开启编辑模式

标签3: "账户权限" (仅管理员可见)
- 账户状态: 激活/禁用 (开关)
- 系统角色: 下拉多选
- 最后登录时间: 只读显示

标签4: "操作记录"
- 时间轴形式
- 显示所有信息变更记录、状态变更记录
- 记录内容: 时间、操作人、变更内容

**顶部操作按钮:**
- 主要: "保存" "编辑"
- 次要: "发送邮件" "重置密码" (管理员)
- 危险: "离职" "删除" (红色,需二次确认)

### 页面3: 新增/编辑员工弹窗

**弹窗布局:**
- 标题: "新增员工" / "编辑员工信息"
- 宽度: 800px
- 步骤式表单: 分3步

**步骤1: 基本信息 (必填)**
- 员工编号: 自动生成 (可修改)
- 姓名: 文本输入
- 性别: 单选 (男/女)
- 联系电话: 手机号输入 + 格式验证
- 邮箱: 邮箱输入 + 唯一性验证
- 部门: 树形下拉选择
- 职位: 下拉选择
- 入职日期: 日期选择器

**步骤2: 详细信息 (可选)**
- 英文名、出生日期、办公位置、头像上传、紧急联系人

**步骤3: 确认提交**
- 汇总显示所有填写信息
- 确认按钮: "提交并创建账户"

## 模块二: 部门架构管理系统

### 页面1: 组织架构图页

**页面布局:**
- 顶部: 页面标题 + 操作按钮
- 中间: 组织架构树形图 (主区域)
- 右侧: 选中部门的详情面板 (可收起)

**顶部操作区:**
- 主按钮: "新增部门" "新增子部门"
- 次要: "导出架构图" "展开全部" "折叠全部"
- 搜索: 快速查找部门

**组织架构树形图 (核心区域):**
- 采用树状图布局,横向展示
- 节点样式:
  - 部门名称 (主文本)
  - 部门人数徽章 (右上角小标签)
  - 负责人头像 (圆形小头像,hover显示姓名)
  - 部门图标
- 交互:
  - 点击节点: 高亮选中,右侧显示详情
  - 双击节点: 展开/折叠子部门
  - 右键菜单: 新增子部门、编辑、删除、移动
- 样式:
  - 使用连接线显示层级关系
  - 不同层级使用不同颜色区分
  - 支持缩放 (滚轮) + 拖拽平移

**右侧详情面板:**
面板标题: "部门详情"
- 部门名称 (可编辑)
- 部门编号
- 上级部门 (链接)
- 部门负责人: 头像 + 姓名 + 联系方式
- 部门人数: 超链接,点击跳转员工列表 (已筛选该部门)
- 成立时间
- 部门描述 (多行文本)
- 操作按钮: "编辑部门" "查看部门成员"

### 页面2: 部门列表页

**页面布局:**
- 顶部: 搜索 + 筛选 + 视图切换
- 中间: 数据表格 / 树形表格

**视图1: 树形表格视图**
列设置:
| 列名 | 说明 |
|------|------|
| 部门名称 | 树形缩进显示层级,显示部门图标 + 名称 |
| 部门编号 | |
| 部门负责人 | 头像 + 姓名 |
| 部门人数 | 可点击,跳转员工列表 |
| 部门级数 | 数字显示 |
| 成立时间 | |
| 操作 | 编辑/删除/新增子部门 |

**视图2: 平板卡片视图**
- 每个部门一个卡片
- 显示部门图标、名称、负责人、人数统计
- 卡片颜色根据部门类型区分

### 页面3: 部门详情弹窗

**弹窗布局:**
- 左侧: 表单区
- 右侧: 部门成员预览列表

**左侧表单字段:**
- 部门名称 (必填)
- 部门编号 (自动生成)
- 上级部门 (树形下拉选择,可选)
- 部门负责人 (员工下拉选择)
- 成立时间 (日期选择器)
- 部门描述 (多行文本框)
- 部门图标 (图标选择器)
- 部门简称

**右侧成员列表:**
- 显示该部门下所有员工
- 列表项: 头像 + 姓名 + 职位
- 操作: "移出部门" (红色叉号按钮)
- 底部: "添加成员"按钮

**底部操作按钮:**
- "保存" "取消" "删除部门" (红色)

## 模块三: 公共资源台账管理系统

### 页面1: 资产列表页

**页面布局:**
- 顶部: 页面标题 + 视图切换 + 操作按钮
- 左侧: 筛选面板
- 中间: 数据内容区

**顶部操作区:**
- 主按钮: "新增资产" "资产入库"
- 搜索框: 按资产名称/编号搜索
- 视图切换: 表格 / 看板 / 画廊
- 批量操作: "批量借出" "批量归还" (选中后显示)

**左侧筛选面板:**
- 资产类别: 电子设备 / 办公家具 / 图书 / 其他 (多选)
- 资产状态: 库存中 / 使用中 / 借出 / 维修中 / 报废 (多选)
- 保管人: 员工下拉搜索
- 存放位置: 文本输入
- 购置时间范围: 日期区间
- 价值范围: 金额区间滑块

**表格视图 (默认):**
列设置:
| 列名 | 说明 |
|------|------|
| 选择 | 复选框 |
| 资产图片 | 缩略图 60x60px |
| 资产编号 | 超链接,点击查看详情 |
| 资产名称 | 主文本 |
| 资产类别 | 标签样式 |
| 品牌/型号 | |
| 资产状态 | 彩色标签 (库存中=绿色,使用中=蓝色,借出=橙色,维修中=红色,报废=灰色) |
| 保管人/使用人 | 头像 + 姓名 |
| 存放位置 | |
| 购置日期 | |
| 当前价值 | 金额显示,带货币符号 |
| 操作 | 借出/归还/编辑/更多 |

**看板视图:**
- 按资产状态分栏显示 (5列)
- 每张卡片显示:
  - 资产图片 (顶部)
  - 资产名称 (标题)
  - 资产编号 (副标题)
  - 保管人 (头像+姓名,如果有)
  - 归还倒计时 (借出状态显示剩余天数,红色标红过期)
- 支持卡片拖拽: 拖拽到不同状态列即可变更状态
- 卡片颜色根据资产类别区分

**画廊视图:**
- 网格布局,每张卡片 300x300px
- 大图展示资产图片
- 显示资产名称、编号、状态标签
- hover显示快捷操作按钮

### 页面2: 资产详情页

**页面布局:**
- 顶部: 返回按钮 + 资产名称 + 操作按钮
- 左侧: 资产图片展示区 (40%宽度)
- 右侧: 详细信息标签页 (60%宽度)

**左侧图片区:**
- 大图展示: 600x600px
- 图片下方: 资产状态大标签
- 支持图片放大查看 (点击灯箱效果)
- 支持多图轮播 (如果有)

**右侧标签页:**

标签1: "基本信息"
- 资产编号 (大字体)
- 资产名称 (大字体加粗)
- 资产类别: 下拉选择
- 品牌/型号: 文本输入
- 资产状态: 下拉选择 (彩色选项)
- 存放位置: 文本输入
- 购置日期: 日期选择器
- 购置金额: 金额输入
- 当前价值: 只读显示 (自动计算折旧)
- 维护记录: 多行文本框

标签2: "使用记录"
- 时间轴形式
- 记录: 借出/归还/维修记录
- 每条记录显示: 时间、操作人、操作类型、备注

标签3: "借还管理"

  - 如果"借出": 显示保管人信息、借出日期、预计归还日期、归还倒计时 (红色标红过期),"归还"按钮

  - 借出日期: 日期时间选择器 (默认当前时间)
  - 预计归还日期: 日期选择器
  - 借用说明: 多行文本
  - "确认借出"按钮

**顶部操作按钮:**
- "编辑资产" "打印标签" "借出" "归还" "报废" (红色)

### 页面3: 新增/编辑资产弹窗

- 标题: "新增资产" / "编辑资产"

- 部门资产分布图: 各部门借用资产数量 (横向柱状图)

## 通用设计规范

### 颜色系统
**主色调:**
- 主色: #1890FF (蓝色) - 主要操作按钮
- 成功: #52C41A (绿色) - 已通过、库存中、在职
- 警告: #FAAD14 (橙色) - 借出、试用期内、待审批
- 错误: #F5222D (红色) - 已驳回、报废、离职、超期
- 信息: #13C2C2 (青色) - 信息提示

**中性色:**
- 标题: #262626 (深灰)
- 正文: #595959 (中灰)
- 辅助文本: #8C8C8C (浅灰)
- 边框: #D9D9D9
- 背景灰: #F5F5F5

### 字体系统
- 标题: 18px-24px, 加粗
- 正文: 14px, 常规
- 辅助文本: 12px, 常规
- 表格表头: 14px, 加粗
- 表格内容: 14px, 常规

### 间距系统
- 页面边距: 24px
- 卡片间距: 16px
- 表单字段间距: 16px
- 按钮间距: 8px

### 圆角系统
- 按钮: 4px
- 输入框: 4px
- 卡片: 8px
- 头像: 50% (圆形)

### 阴影系统
- 卡片阴影: 0 2px 8px rgba(0,0,0,0.08)
- 弹窗阴影: 0 4px 16px rgba(0,0,0,0.15)
- hover阴影: 0 4px 12px rgba(0,0,0,0.12)

### 交互规范
- 按钮hover: 背景色加深10%
- 表格行hover: 背景色 #FAFAFA
- 可点击元素: cursor pointer
- 加载状态: 骨架屏 + Loading动画
- 空状态: 插图 + 文字说明 + 操作按钮
- 错误提示: 顶部Toast通知 + 表单项下方红色文字

### 响应式设计
- 桌面端: >= 1440px (主要支持)
- 平板: 768px - 1439px (表格横向滚动,侧边栏折叠)
- 移动端: < 768px (单列布局,抽屉式侧边栏)

---

## 数据交互规范

### 加载状态
- 页面加载: 骨架屏占位
- 按钮加载: 按钮内loading动画
- 表格加载: 表格区域loading动画

### 空状态
- 无数据: 显示空状态插图 + "暂无数据"文字 + 新增按钮
- 搜索无结果: "未找到匹配结果" + 清除筛选按钮

### 错误处理
- 网络错误: 显示重试按钮
- 表单验证错误: 字段下方红色提示 + 聚焦该字段
- 权限错误: 显示权限不足提示 + 联系管理员

### 成功反馈
- 操作成功: 顶部Toast提示 "操作成功"
- 数据保存: Toast提示 "保存成功"
- 删除确认: 二次确认弹窗 + 成功后Toast提示

---

## 权限控制显示

### 管理员视图
- 显示所有操作按钮
- 可编辑所有字段
- 可看到敏感信息
- 可删除数据

### 普通员工视图
- 只读查看基础信息
- 个人可编辑个人信息 (部分字段)
- 不显示删除/敏感操作按钮

### 部门管理员视图
- 可管理本部门员工
- 可查看本部门资产
- 不可跨部门操作

---

## 辅助功能

### 快捷键
- Ctrl/Cmd + F: 聚焦搜索框
- Ctrl/Cmd + N: 新增记录
- ESC: 关闭弹窗
- Enter: 保存表单

### 导出功能
- 支持导出Excel格式
- 支持导出PDF格式 (架构图)
- 导出时支持选择字段

### 打印功能
- 支持打印员工信息表
- 支持打印资产标签
- 打印预览功能
