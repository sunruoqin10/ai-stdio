## 📖 模块简介

资产管理模块用于管理公司的固定资产、办公设备等资源,支持资产的入库、借用、归还、维修、报废等全生命周期管理。

### 实现亮点

**1. 三种视图模式**
- 表格视图: 传统列表展示,支持分页
- 看板视图: 按状态分组,支持拖拽改变状态(vue-draggable-plus)
- 画廊视图: 卡片网格布局,展示资产图片

**2. 完整的Mock数据支持**
- 12个预置资产,涵盖4种类别
- 借还记录数据
- 统计数据预置
- 完整的CRUD API模拟

**3. 自动折旧计算**
- 月折旧率1%
- 购置当月不计折旧
- 折旧到0为止,不会出现负值
- 实时更新当前价值

**4. 智能提醒系统**
- 到期提醒(默认提前3天)
- 逾期检测和标记
- 支持自定义提醒天数

**5. 状态转换验证**
- 严格的状态转换规则
- 防止非法状态变更
- 支持业务流程控制

**6. 丰富的统计功能**
- 6个统计卡片(总资产、总金额、当前价值、借出、维修、报废)
- 资产分类饼图
- 资产状态饼图
- 折旧趋势柱状图
- 借出趋势折线图
- 分类统计表

**7. 工具函数库**
- 折旧计算
- 提醒检测
- 编号生成
- 格式化工具
- 状态验证
- 分组工具
