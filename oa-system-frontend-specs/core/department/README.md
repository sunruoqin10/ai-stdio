## 📖 模块简介

部门管理模块是OA系统的核心基础模块,用于管理公司的组织架构,支持树形结构的部门层级关系(最多5级),提供部门CRUD、组织架构图可视化、部门成员管理等功能。

### 实现亮点

**1. 完整的Mock数据支持**
- 12个预置部门,覆盖5级层级结构
- 部门负责人关联员工数据
- DiceBear头像自动生成
- 完整的CRUD操作模拟

**2. 强大的树形处理能力**
- 扁平数组与树形结构相互转换
- 节点路径、层级自动计算
- 子部门递归查询
- 移动验证(防止循环引用)

**3. 统计功能**
- 实时统计部门人数(从员工表)
- 按层级分析部门分布
- 有负责人部门统计
- 总员工数统计(去重)

**4. 批量操作**
- 批量删除部门
- 批量更新部门层级
- 批量更新部门人数

**5. 导出功能**
- 支持导出部门列表为Excel
- 可导出树形或扁平结构
