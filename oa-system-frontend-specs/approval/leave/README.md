## 📖 模块简介

实现员工请假申请、审批、统计的全流程管理,支持多级审批、自动计算时长、余额提醒等功能。

### 实现亮点

**1. 完整的Mock数据支持**
- 6个预置请假申请(涵盖各种状态)
- 4个年假余额记录
- 12个节假日数据(2026年)
- 3个审批记录示例
- 完整的CRUD API模拟

**2. 多级审批流程**
- 1级审批: ≤3天(部门负责人)
- 2级审批: 4-7天(部门负责人 + 人事)
- 3级审批: >7天(部门负责人 + 人事 + 总经理)
- 按顺序审批,任一级驳回则整个申请驳回
- 所有级通过才通过

**3. 工作日计算**
- 只计算工作日(周一到周五)
- 自动排除周末
- 自动排除法定节假日
- 支持半天请假(0.5天)
- 支持跨月连续请假

**4. 年假管理**
- 自动根据工龄计算年假额度(<1年:0天, 1-9年:5天, 10-19年:10天, ≥20年:15天)
- 年假余额验证
- 年假自动扣减
- 年假余额预警(≤3天)

**5. 丰富的工具函数**
- 日期时间格式化
- 请假类型/状态转换
- 时长计算(工作日)
- 状态判断(可撤销/可编辑/可删除/可重新提交)
- 年假额度计算
- 工龄计算
- 审批流程判断
