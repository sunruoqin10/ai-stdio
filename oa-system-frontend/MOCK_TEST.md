## 🚀 快速测试

1. **启动项目**:
   ```bash
   npm run dev
   ```

2. **访问登录页**:
   ```
   http://localhost:5173/login
   ```

3. **输入账号密码**:
   - 账号: `admin`
   - 密码: `Admin123`

4. **点击登录**:
   - ✅ 登录成功
   - ✅ 自动跳转到首页
   - ✅ 显示顶部菜单
